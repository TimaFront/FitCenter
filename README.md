## About the Project :
Мой первый пробный FullStack проект. По времени ушло примерно месяц, с учетом того, что нужно было изучать BackEnd часть и разбираться с фреймворком C# ASP.NET CORE MVC. Для базы данных использовал PostgreSQL. Написал дизайн на "Figma", логотип делал в Adobe Illustrator. Примечатльно, что всем этим я раньше не занимался, кроме FrontEnd ом. Эту работу я представлял как курсовую и сдал ее на отлично в максимально короткие сроки. Для запуска потребуется и БД
